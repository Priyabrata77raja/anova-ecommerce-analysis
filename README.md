# anova-ecommerce-analysis# ANOVA E-commerce Analysis

## 📊 Objective
Analyze whether different age groups have different purchasing behavior.

## 🛠 Tools Used
- Python
- Pandas
- SciPy
- Matplotlib

## 🧪 Method
Applied One-Way ANOVA test to compare means.

## 📈 Result
Identified whether age groups significantly affect purchase amount.
